Database for speech processing
1. Speaker dependent isolated word recognition in English: (SDIWR) this database contains 10000 wav files. There are 1000 words recorded in the voice of a single speaker each word uttered 10 times. The files are named as follows. MSB digit indicates the utterance number; next 3 digits indicate the word number. For example wav file with name as 1234 will mean the 1st utterance for word number 234.
Link- 2. Speaker independent isolated word recognition in English: (SIIWR) this database contains 10000 wav files. There are 100 words recorded in the voice of 10 speakers each word uttered 10 times. The files are named as follows. MSB digit indicates the speaker number, the next digit indicates the utterance number; next 2 digits indicate the word number. For example wav file with name as 1234 will mean the 1st speaker, utterance number 2 and utterance for word number 34.
Link- 3. Emotional speech database in English: this database contains 10000 wav files. There are 10 sentences recorded in the voice of 10 speakers each sentence uttered 10 times. These sentences are recorded in 5 different emotions, happy(H), angry(A), sad(D), fear(F), neutral(N) and surprise(S) emotion. The files are named as follows. MSB digit indicates the emotion category, the next digit indicates speaker number, the next digit indicates sentence number and last digit indicates the utterance number, For example wav file with name as D1234 will mean the sad utterance for speaker number 1, 2nd utterance and last digit 4 indicates 4th utterance.
Link Payment: Each database can be downloaded from the respective link by subscribing to the database after login and by paying Rs. 5000 using the payment GATEway.

Emotion Detection from speech-(Wakonnati): We have considered four emotions, Happy, angry, sad and Neutral. The system is tested on the Berlin Database for emotional speech. The recognition accuracy is as follows: I) Angry- 99%, ii) Happy -94%, iii) Neutral- 91% and iv) Sad- 81%. This is probably the highest accuracy reported on Berlin Database. Conversion of Neutral Speech to Emotional Speech - (Wagwilasini): We take the neutral utterance from the user or the system may use already stored sentences in neutral mode. The user is asked to select the emotional effects from the following. i) start and end effect for the sentence ii) time gap adjustment iii) effect of question mark iv) effect of exclamatory mark v) Effect of fast Speech vi) Sad emotion vii) Happy emotion viii) Angry emotion ix) Normal speaking style x) Context dependent adjustment xi) lifting of the pitch etc. This is already patented Context Dependent Emotional Speech Synthesis- (Wagwikasini): The system records the context dependent spoken words from the user and stores small parts of the words. The context for each word decides the small parts of words to be concatenated to form another word. The system synthesizes different words and generates the sentence. The emotional effects stated in Wagwilasini are added to synthesize the emotions. The special marks in the text will guide the system to add a particular emotion effect for some words in the sentence. The synthesized sentence is the emotional speech as required by the user. The synthesis is personalized for each user recorded voice. Work is patented.
